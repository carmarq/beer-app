Conceptual Model

Entities & Attributes:

Profile

  • profileId Binary(16) NOT NULL (PRIMARY KEY)
  • profileAbout VarChar(140)
  • profileAddressLine1 VarChar(96) NOT NULL
  • profileAddressLine2 VarChar(96) NOT NULL
  • profileCity VarChar(48) NOT NULL
  • profileEmail VarChar(128) NOT NULL
  • profileHash Char(97) NOT NULL
  • profileImage VarChar(255)
  • profileName VarChar(64) NOT NULL
  • profileState Char(2) NOT NULL
  • profileUsername Varchar(48), NOT NULL
  • profileUserType Char(1) NOT NULL
  • profileZip VarChar(10) NOT NULL

Beer

  • beerId Binary(16), NOT NULL (PRIMARY KEY)
  • beerProfileId Binary(16), NOT NULL (FOREIGN KEY)
  • beerAbv Decimal(66), NOT NULL
  • beerDescription VarChar(1024)
  • beerIbu VarChar(3)
  • beerName VarChar(128), NOT NULL

Style

  • styleId Binary(16), NOT NULL (PRIMARY KEY)
  • styleType TinyInt(unsigned), NOT NULL

Beer Style

  • beerStyleBeerId Binary(16), NOT NULL (FOREIGN KEY)
  • beerStyleStyleId Binary(16), NOT NULL (FOREIGN KEY)

Vote

  • voteBeerId Binary(16), NOT NULL (FOREIGN KEY)
  • voteProfileId Binary(16), NOT NULL (FOREIGN KEY)
  • voteValue TinyInt, NOT NULL
